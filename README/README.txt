Th project is made on Django framework along with javascript,HTML,CSS,Bootstrap.

The website is not static entirely.
The db.sqlite file is important as it contains the data.

The data is added from backend admin panel and Is reflected on the front
Backend is used so as to avoid same code in the html files 
data will automatically parsed from backend. 

To  run the project python should be installed along with pip 

Commands required to run django

open cmd inside golden_ratio_project folder

run the following commands

pip install django
python manage.py runserver

the server is now up and running

to access website go to 127.0.0.1:8000

to access backend admin panel go to 127.0.0.1:8000/admin
the credentials of admin panel is

username : admin
password : admin
